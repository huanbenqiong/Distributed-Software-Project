# Distributed-Software-Project

分布式软件原理与技术课程项目 - 基于 Java 的简易分布式通信节点原型。

## 🎯 项目目标
本项目旨在实现一个轻量级的分布式节点通信框架，为后续的分布式存储、负载均衡或 RPC（远程过程调用）机制提供底层网络支持。

## 🛠️ 技术栈
- **核心语言**：Java (JDK 11+)
- **网络通信**：Java 原生 Socket / NIO (规划中)
- **数据序列化**：JSON / 基础字节流
